# MoodTrack — 오늘 감정에 맞는 플레이리스트 AI 추천 서비스

> 지금 기분을 한 문장으로 입력하면, LLM이 감정을 분석해 Spotify/YouTube 플레이리스트 링크를 즉시 제공해주는 웹 서비스

---

## 서비스 주제 & 한 줄 소개

**MoodTrack**은 사용자가 현재 감정을 텍스트로 입력하면 LLM이 감정 상태를 분류하고, 해당 감정에 맞는 음악 플레이리스트를 Spotify/YouTube 링크로 즉시 추천해주는 웹 기반 서비스입니다.

---

## 문제 정의

### 현황 (As-Is)

- 국내 음악 스트리밍 사용자의 약 73%가 하루 1회 이상 플레이리스트를 직접 검색함 (2024 멜론 리서치)
- 기존 스트리밍 서비스(Spotify, 멜론, 유튜브 뮤직)의 추천 알고리즘은 **청취 이력 기반**이어서, 오늘 처음 느끼는 감정 변화에는 즉각 반응하지 못함
- "지금 이 기분에 딱 맞는 음악"을 찾기 위해 사용자는 평균 4~7분 동안 검색/스크롤을 반복함

### 원인

- 스트리밍 서비스는 **과거 데이터 기반 추천**만 제공하며, 현재 감정의 언어적 맥락을 이해하지 못함
- 플레이리스트 태그("운동", "공부", "드라이브")가 너무 단순하여 복합적 감정 상태를 반영하지 못함
- 사용자가 자신의 감정을 음악 키워드로 변환해야 하는 인지 부담이 존재함

### 해결안

- 사용자가 "오늘 발표 망했고 비도 오고 그냥 멍하니 있고 싶어" 와 같이 **자연어로 감정을 입력**하면
- LLM이 감정 벡터(슬픔 70%, 무기력 20%, 위로 필요 10%)로 분석하고
- 해당 감정 조합에 맞는 Spotify/YouTube 플레이리스트 3개를 **즉시 링크로 제공**

## As-Is / To-Be 비교

| 항목 | As-Is (현재) | To-Be (MoodTrack) |
|------|-------------|------------------|
| 음악 탐색 시간 | 평균 4~7분 검색/스크롤 | 감정 입력 후 5초 이내 추천 |
| 추천 기반 | 과거 청취 이력 | 현재 감정 텍스트 실시간 분석 |
| 감정 표현 방식 | 단순 태그("운동", "슬픔") 선택 | 자연어 문장 자유 입력 |
| 복합 감정 반영 | 불가능 | 감정 벡터로 다차원 분석 가능 |
| 플레이리스트 다양성 | 알고리즘이 익숙한 곡 반복 추천 | 감정에 따른 새로운 큐레이션 제공 |

### 페인포인트 (불편함)

1. **탐색 피로**: 논문 작업 중 집중력이 끊길 때마다 "지금 내 기분에 맞는 음악"을 찾으려고 Spotify와 YouTube를 번갈아 검색하는 데 5분 이상 소비함. 정작 음악보다 탐색에 더 지침.

2. **추천 알고리즘의 편향**: Spotify가 최근 1주일 청취 이력 기반으로만 추천해서, 오늘처럼 갑자기 "조용하고 무거운 재즈"가 듣고 싶을 때 평소에 듣던 팝 위주로만 추천됨. 기분 변화를 반영하지 못함.

3. **감정→키워드 변환 부담**: 기분이 미묘하거나 복합적일 때(예: 기쁘지만 뭔가 허전함) 이를 Spotify 검색창에 어떤 단어로 입력해야 할지 막막함. 감정을 음악 태그로 번역하는 과정 자체가 스트레스.

4. **반복 노출 문제**: 같은 플레이리스트를 이미 100번 이상 들어서 새로운 음악을 발견하고 싶지만, 알고리즘이 항상 비슷한 아티스트만 추천해 음악 취향이 넓어지지 않음.

### 니즈 (사용자가 원하는 것)

- 지금 이 순간의 감정을 말로 설명하면 **5초 이내에** 그 감정에 맞는 플레이리스트 링크를 받고 싶다
- 복잡한 감정(예: "긴장되지만 설레는 발표 전날 밤")도 **별도 번역 없이** 그대로 입력할 수 있어야 한다
- 기존에 몰랐던 아티스트나 장르의 음악을 발견할 수 있는 **새로운 큐레이션**을 원한다
- 음악 탐색을 위해 현재 작업(공부, 업무)을 **30초 이상 중단하지 않아도** 되는 흐름을 원한다

## 핵심 기능 (MVP)

1. **감정 텍스트 입력창**: 자유 형식 자연어 입력 (최대 200자)
2. **LLM 감정 분석**: 입력된 텍스트를 감정 벡터로 분류 (예: 슬픔 60%, 위로 30%, 고요함 10%)
3. **플레이리스트 추천**: Spotify/YouTube 플레이리스트 3개 링크 + 추천 이유 한 줄 제공
4. **감정 히스토리**: 날짜별 감정 기록 저장 (선택 기능)
